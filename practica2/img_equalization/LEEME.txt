Ecualización de la imagen digital.

Se han resuelto todos los apartados propuestos, tanto los test como el programa principal.

Para aplicar una equalizacion global:
    ./build/img_equalization <input> <output>

    Ejemplo imagen blanco y negro:
        ./build/img_equalization data/radiografia.png data/out.png
    Ejemplo imagen a color:
        ./build/img_equalization data/cells.png data/out.png

Para aplicar una equalizacion local (radio de ventana 25):
    ./build/img_equalization <input> <output> -r=25

    Ejemplo imagen blanco y negro:
        ./build/img_equalization data/radiografia.png data/out.png -r=25
    Ejemplo imagen a color:
        ./build/img_equalization data/cells.png data/out.png -r=25

Para aplicar una equalizacion bimodal:
    ./build/img_equalization <input> <output> -m

    Ejemplo imagen blanco y negro:
        ./build/img_equalization data/radiografia.png data/out.png -m
    Ejemplo imagen a color:
        ./build/img_equalization data/cells.png data/out.png -m

Para aplicar una equalizacion local y bimodal:
    ./build/img_equalization <input> <output> -r=25 -m

    Ejemplo imagen blanco y negro:
        ./build/img_equalization data/radiografia.png data/out.png -r=25 -m
    Ejemplo imagen a color:
        ./build/img_equalization data/cells.png data/out.png -r=25 -m
